EMA-Kotlin Playground

Explore Expressive and Modular Activation in Kotlin. This page includes interactive snippets inspired by EMA-js examples, now adapted for Kotlin. The browser snippets are intentionally lightweight; the production runtime is in this repository.

Examples correspond to the EMA-Kotlin approach and repository tests. For the full runtime API, use the library source and README. EMA-Kotlin is inspired by and extends the ideas in: Leger, P., Cardozo, N., & Masuhara, H. (2023). An expressive and modular layer activation mechanism for Context-Oriented Programming. Information and Software Technology, 156, 107132. https://doi.org/10.1016/j.infsof.2022.107132.
